# Stimuler · Language Settings prototype

Interactive design prototype for Stimuler's video-lesson language controls, exit/reactivation states, and the settings page.

Vanilla HTML/CSS/JS. Use the **Flows** chips (right rail) to explore:

- **Video-lesson controls** — roadmap → lesson → morphing glass control dock → switch-to-Spanish warning/loader → reading state
- **Exit** / **Reactivation** — the exit-sheet and reactivation flows over the circular-Sarah screen
- **Settings page** — revamped profile/settings with the 3-case language model (Mother tongue · App language · Video language), switchable via the nested Case chips
